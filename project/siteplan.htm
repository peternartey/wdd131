Student Dashboard

Student Dashboard Logo

Overview: This site will provide a clean and simple dashboard where students can track assignments, grades, and schedules. It will feature a homepage summary and subpages for different course categories.

Domain Name: student-dashboard.netlify.app

Site Purpose

The student dashboard helps students manage their academic responsibilities in one place. It includes assignment deadlines, GPA tracking, personal goals, and useful academic resources.

Scenarios

  • What assignments are due this week?
  • How am I performing in my courses?
  • Where can I upload or download class notes?

Color Scheme

  • Primary Color (Navy Blue - #2C3E50): headers, navigation bar, and footer
  • Accent Color (Light Teal - #1ABC9C): buttons, highlights, and callouts
  • Background Color: #FAFAFA

Typography

  • Roboto: Used for all text for a clean and modern look.
